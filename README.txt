欢迎大家使用PCC~

【简介】
PvZ Controller by CPP（简称PCC）是由him君制作的一个用于Steam植僵年度版的键控工具。（这也太简洁了吧啊喂！）

【功能】
PCC拥有大部分键控工具拥有的功能，包括自动发炮、PreJudge、自动选卡等。

【使用说明】
使用之前请先仔细阅读以下两篇指南：
准备&安装篇：https://blog.xiangxinzhe.com/2022/07/12/zhijiangjiankonggongjupccshiyongzhinanzhunbeianzhuangpian/
使用篇：https://blog.xiangxinzhe.com/2022/07/29/zhijiangjiankonggongjupccshiyongzhinanshiyongpian/
